Chrome Shortcuts

...and some other browsers...
A majority of my computer usage is in a web browser, specifically Google Chrome. That's why the most popular browser (as of this writing) has spawned its own operating system and hardware: Chrome OS, Chromebooks, & Chromeboxes.

I like saving time...


...so browser shortcuts is a good place to start. I listed out essential hotkeys or shortcuts pretty much every Chrome user should know.

Open Link in a New Tab

  • Ctrl + Left Click on link
  • Middle Click on link

Open a New Tab

  • Ctrl + T

Close Tab

  • Ctrl + W

Open Prior Closed Tab

  • Ctrl + Shift + T

History

  • Ctrl + H

Downloads

  • Ctrl + J

There are always more hotkeys. These are the "basics". There still is a little too much finger gymnastics for my liking but I'll address that another time.
